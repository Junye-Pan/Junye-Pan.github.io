# Posts

Use one directory per post or project writeup:

```text
posts/<slug>/
  index.html
  source.md
  assets/
    report.pdf
    thumb.webp
    images/
    source-plots/
```

Keep site-wide files in the root `assets/` directory, such as shared CSS and the
profile image. Keep post-specific reports, thumbnails, plots, source markdown, and
raw plot exports inside the corresponding post directory.
